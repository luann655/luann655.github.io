Nuestro Proyecto

GasGuard es un sistema automatizado que utiliza sensores de gas para detectar la presencia de gases nocivos en el hogar. Cuando se detecta una concentración peligrosa, el sistema activa automáticamente mecanismos para abrir ventanas y puertas, facilitando la ventilación y reduciendo el riesgo de intoxicación.

El sistema se basa en la lectura continua de los niveles de gas mediante sensores especializados. Cuando estos sensores detectan una concentración que supera un umbral predefinido, el Arduino activa el módulo de relé, que a su vez activa los servomotores para abrir las ventanas y puertas.

Además, GasGuard incluye un sistema de alerta con buzzer, permitiendo que la casa este segura, incluso cuando no están en casa.

Detalles del Proyecto

Componentes del Proyecto:

Arduino Uno

Microcontrolador que actuará como el cerebro del sistema, gestionando las señales y controlando otros componentes.

Sensor de Gas (MQ-2 o MQ-7)

Dispositivo encargado de detectar la presencia de gases peligrosos como metano, propano y monóxido de carbono en el ambiente.

Módulo de Relé

Permite activar o desactivar dispositivos eléctricos, como servomotores, para abrir ventanas y puertas cuando se detectan niveles peligrosos de gas.

Servomotores

Utilizados para abrir y cerrar automáticamente las puertas cuando se activa el sistema de relé.

Fuente de Alimentación

Proporciona la energía necesaria para alimentar el Arduino y los demás componentes del sistema.

Cables y Protoboard

Se utilizan para realizar las conexiones físicas entre los distintos componentes electrónicos del proyecto.

Software de Programación (Arduino IDE)

Entorno de desarrollo que permite programar el comportamiento del sistema y las interacciones entre los componentes.

Objetivos

  • Detectar la presencia de gases peligrosos en el ambiente de manera eficiente y precisa.
  • Automatizar la apertura de ventanas y puertas para mejorar la ventilación en caso de detección de gas.
  • Aumentar la seguridad en el hogar mediante la implementación de un sistema de alerta temprana.
  • Educar sobre la importancia de la detección de gases y la seguridad en el hogar.

Funcionamiento

El sistema se basa en la lectura continua de los niveles de gas mediante el sensor de gas. Cuando el sensor detecta una concentración de gas que supera un umbral predefinido, el Arduino activa el módulo de relé, lo que a su vez activa los servomotores para abrir las ventanas y puertas. Además, el sistema puede incluir una alarma o buzzer para alertar a los ocupantes sobre la detección de gas.
